# 使用指南

## 快速开始

### 1. 打开工具
在浏览器中打开 `index.html` 文件

### 2. 基本设置
- **地图尺寸**: 设置宽度和高度（推荐1200x1200）
- **格子大小**: 设置每个格子的边长（推荐100像素）

### 3. 生成地图
点击"生成地图"按钮或按 `G` 键

## 操作说明

### 视角控制
- **俯视模式**: 点击"0° (俯视)"或按 `1` 键
- **倾斜模式**: 点击"45° (左旋)"或按 `2` 键

### 缩放操作
- **放大**: 点击"放大"按钮或按 `+`/`=` 键
- **缩小**: 点击"缩小"按钮或按 `-` 键
- **重置**: 点击"重置缩放"按钮或按 `0` 键

### 地图导航
- **选择格子**: 左键点击任意格子
- **拖拽移动**: 右键按住并拖拽
- **悬停查看**: 鼠标悬停显示格子边界

### 数据导出
- **导出数据**: 点击"打印数据"按钮或按 `S` 键
- **文件格式**: 自动生成 `.txt` 文件

## 键盘快捷键

| 按键 | 功能 |
|------|------|
| `G` | 生成地图 |
| `+` / `=` | 放大 |
| `-` | 缩小 |
| `0` | 重置缩放 |
| `1` | 0度视角 |
| `2` | 45度视角 |
| `S` | 导出数据 |

## 格子信息

点击格子后可查看：
- 格子编号
- 行列位置
- 圈层信息
- 四个角的坐标点
- 当前倾斜角度

## 输出数据格式

导出的文本文件格式：
```
格子编号:左下角X;左下角Y|左上角X;左上角Y|右上角X;右上角Y|右下角X;右下角Y
1:0;0|0;99|99;99|99;0
2:100;0|100;99|199;99|199;0
...
```

## 常见问题

### Q: 地图生成失败怎么办？
A: 检查输入参数是否在有效范围内，地图尺寸应在100-2000像素，格子大小应在10-200像素。

### Q: 如何精确查看格子坐标？
A: 点击任意格子，右侧信息面板会显示详细的坐标信息。

### Q: 导出的数据文件在哪里？
A: 文件会自动下载到浏览器默认下载目录，文件名包含时间戳和角度信息。

### Q: 支持哪些浏览器？
A: 支持Chrome、Firefox、Safari、Edge等现代浏览器的较新版本。
